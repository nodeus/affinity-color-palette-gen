# Changelog

## [9.1.0] - 2026-06-28

### Fixed
- Palette now generates for the current active page/artboard instead of always the first one.
- Replaced `doc.spreads.first` with `doc.currentSpread`.

## [9.0.0] - 2026-06-28

### Added
- Extracts fill, stroke, and gradient colors from selected nodes or entire spread.
- Displays swatches with RGB, CMYK, HSL, and HEX values, plus gradient stops.
- Groups outputs by fills, strokes, and gradients.
